# Rock Paper Scissors 🎮

A simple command-line Rock Paper Scissors game built with Python.

## How to Play
- Run the script
- Enter 0 for Rock, 1 for Paper, 2 for Scissors
- The computer picks randomly
- See who wins!

## Features
- ASCII art visuals for each choice
- Random computer selection
- Win / Lose / Draw detection

## Run
```bash
python Rock_Paper_Scissors_Game.py
```
